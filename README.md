# 奥通线缆 PbootCMS 仿站

本仓库基于 PbootCMS V3.2.14 搭建，前台模板位于 `template/aotong`，用于复刻电线电缆企业站风格。

## 已完成内容

- 已集成 PbootCMS 程序和 SQLite 默认数据库。
- 已切换站点主题为 `aotong`。
- 已建立产品中心三大分类：低压电缆、高压电缆、电线。
- 产品列表页支持三分类切换。
- 产品详情页上半部分读取后台上传内容：标题、主图、组图、简介、分类扩展字段、正文内容。
- 产品详情页下半部分为固定板块：产品详情、产品参数、厂家优势、热品推荐。
- 已做电脑端和手机端自适应样式。

## 当前访问地址

本机服务运行在 `8088` 端口时，推荐用这一套地址：

- 前台首页：`http://10.191.100.43:8088/`
- 后台地址：`http://10.191.100.43:8088/admin.php`
- 默认账号：`admin`
- 默认密码：`123456`

说明：`127.0.0.1` 和 `localhost` 只是电脑本机访问方式，不是另一套后台。手机查看时请使用上面的局域网地址。

请上线后第一时间修改后台密码，并按正式域名申请/配置 PbootCMS 授权码。

## 内容维护

产品内容在后台的“内容管理”里维护：

- 栏目：产品中心
- 分类：低压电缆 / 高压电缆 / 电线
- 可替换字段：产品标题、缩略图、轮播组图、产品详情正文、产品类型等扩展字段

客户后续替换图片时，建议通过后台上传；模板中固定展示板块可在 `template/aotong/html/product.html` 调整。

## 部署要求

- PHP 7.0+ 或兼容 PbootCMS V3.2.14 的 PHP 环境
- 支持 SQLite，或按 PbootCMS 官方方式切换 MySQL
- 将仓库文件上传到网站根目录后访问首页即可

## 本地预览

本地开发服务使用仓库内的 `dev-router.php` 启动。`dev-router.php` 仅用于 PHP 内置服务器预览，正式部署到 Apache/Nginx/PHP 空间时不需要单独配置它。

已用本地 PHP + SQLite 实际请求验证首页、产品分类页、产品详情页均可正常渲染。
